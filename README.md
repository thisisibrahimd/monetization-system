# Monetization System

A web app to monetization a discord server with paypal subscriptions

## Installation

Clone the repo

```bash
git clone https://github.com/thisisibrahimd/monetization-system.git
```

Install packages

```bash
npm install
```

## Usage

Fill out the apporpriate details in ```config.js```

Start the dev server

```bash
npm run dev
```
